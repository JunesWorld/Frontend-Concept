# Frontend-concept

## 용어
- Grave : ` (1번 왼쪽)
- Tilde : ~
- At sign : @
- Caret : ^ (6)
- Ampersand : &
- Asterisk : *
- Verticalbar : | or 원
- Brace : { }
- Bracket : [ ]
- Parenthesis : ( )
- Angle bracket : < >

## 단축키
- 사이드바 열고 닫기 : Cmd + B
- 파일 검색 : Cmd + P
- 모든 명령 : Cmd + shift + P
- 닫기 :Cmd + W
- 바꾸기 : Cmd + alt + f
- 줄 이동 : Alt + up/down
- 줄 복사 : Alt + shift + up/down
- 내어쓰기 : Shift + Tab
- 자동 setting : Cmd + alt + L
- 옆 파일로 이동 : Cmd + shift + [
- 화면 분할 : Cmd + \
- 전체 저장 : Alt + cmd + S

## Settings
- Korean Language : 한국어로 변경 지원
- Beautify(Prettier) : 자동 정렬 기능
> 기능 기여도 -> HookyQR 복사
> Code -> 기본 설정 -> 바로가기 key -> Beautify Selection
> Auto Rename Tag -> Alt + Cmd + L
- Live Server
> HTML File에서만 작동
> 우클릭 -> Open with Live Server
- 공백 2 설정
> Code -> 기본설정 -> 설정 -> Editor -> Tab Size 2

## VSCode
- Project 단위 = Folder(Directory)
  * File – Open – 폴더 선택

## HTML(Hyper Text Markup Language) : 기획자(구조)
- 페이지의 제목, 문단, 표, 이미지, 동영상 등 웹의 구조를 담당
- 쉽게 말해 링크가 걸려있는 파일을 만들 수 있는 곳
- ! + enter : HTML 기본 구조 생성
- <!DOCTYPE html> : 문서(페이지)의 HTML 버전을 지정
- <html> : 문서의 전체 범위
  - <html lang=”ko”> : 한국어 설정
- <head> : 문서의 정보를 나타내는 범위(제목, 설명, 스타일(CSS))
